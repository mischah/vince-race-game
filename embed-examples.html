Race Game Embedding Examples

This page demonstrates different ways to embed the race game in your website.

Fixed Width (400x600)

<iframe src="https://mischah.github.io/vince-race-game/" width="400" height="600"></iframe>

Fixed Height (400px)

<iframe src="https://mischah.github.io/vince-race-game/" width="100%" height="400"></iframe>

Wide Format (300px height)

<iframe src="https://mischah.github.io/vince-race-game/" width="100%" height="300"></iframe>

Small Square (300x300)

<iframe src="https://mischah.github.io/vince-race-game/" width="300" height="300"></iframe>

How to Use

To embed the race game on your website:

  1. Copy the iframe code from one of the examples above
  2. Paste it into your HTML where you want the game to appear
  3. Adjust the width and height as needed
  4. Add the script tag to ensure proper scaling in iframes:
<script src="https://mischah.github.io/vince-race-game/iframe-handler.js"></script>
